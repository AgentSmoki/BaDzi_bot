# Skill: Время, прогнозы, активации

При ответе на временной вопрос используй данные из `[CURRENT_MOMENT]` и `[BAZI_DATA].Ближайшие столпы удачи`. Не выдумывай даты — цитируй из блоков дословно.

## Что анализировать (порядок)

### 1. Иерархия силы влияния (всегда в этом порядке)

1. **Такт удачи (大運)** — фоновая энергия на 10 лет, самое сильное влияние.
2. **Годовой столп (流年)** — энергия года, активирует/блокирует такт.
3. **Месяц (流月)** — тонкая настройка.
4. **День (流日)** — микрособытия.

Тройное взаимодействие (карта + такт + год) → ключевые события года.

### 2. Анализ Такта Удачи

- Определить ствол + ветвь текущего такта (из `[BAZI_DATA].Ближайшие столпы удачи`).
- Оценить через 10 Божеств к Дневной Мастер:
  - Такт Юн Шэнь → благоприятный 10-летний период.
  - Такт Цзи Шэнь → период вызовов, нужны корректировки.
- Проверить взаимодействия такт ↔ натальная карта:
  - Слияние такта со столпом → активация этого столпа (год/месяц/день/час).
  - Столкновение такта с натальной ветвью → перестройка соответствующей сферы.
  - Наказание (刑) — глубокие конфликты, юридика, хирургия.

### 3. Анализ текущего года

Из `[CURRENT_MOMENT].Столпы`:
- Год взаимодействует ОДНОВРЕМЕННО с натальной картой И с текущим тактом.
- Какие Божества активирует год?
- Слияния года с Дневной Мастер → возможны значимые события (партнёр, сделка, переезд).
- Столкновения года с натальными ветвями → нестабильность в соответствующих сферах.

### 4. Резонансы (использовать `[CURRENT_MOMENT].Резонансы натала с текущим моментом` если есть)

- **六合** (гармония) натальной ветви с текущей → активирует столп позитивно. Сферы столпа «звучат».
- **六冲** (столкновение) → возмущает столп. Та сфера в зоне риска.
- Особое внимание: натальная Звезда Шэнь Ша в столпе, который сейчас в резонансе.

### 5. Структуры карты (格局) и тактовый анализ

При сильной структуре (正官格, 七殺格, 食神格 и т.д.):
- Юн Шэнь карты приходит в такте → структура «звучит», период подъёма.
- Цзи Шэнь приходит → структура «разламывается», нужны защитные действия.
- 化氣格 (структура трансформации) → особо чувствительна к ствольным слияниям такта.

### 6. Звёзды и стадии роста

- **12 стадий роста (十二長生)** для Дневной Мастер:
  - Натальная стадия даёт базовый «градус» энергии.
  - Такт может переводить Дневной Мастер из «Императора» (帝旺) в «Болезнь» (病) — это надо озвучить.
- **Пустые ветви (空亡)** натала:
  - Когда такт приносит эти ветви — пустота «включается», нереализованное проявляется.
- Активные Звёзды Шэнь Ша на текущем моменте (из `[CURRENT_MOMENT].Активные звёзды`) — упомянуть, если в резонансе с натальными.

## Если запрос про выбор даты (`[CALENDAR_SELECTION]` присутствует)

Бот уже посчитал топ/анти-список дат с оценками — твоя задача:
1. Назвать **2-3 лучших дня** из top списка с обоснованием через ствол+ветвь.
2. Назвать **2-3 худших дня** из bottom списка как предупреждение.
3. Объяснить, какие элементы дают резонанс с конкретной картой клиента в эти даты.
4. Не выдумывать даты, которых нет в блоках.

## Формат ответа по прогнозу

1. **Общая энергия периода** — одно предложение.
2. **Главная тема/вызов** — через ключевое Божество такта/года.
3. **Возможности** — конкретные сферы (карьера/отношения/здоровье/финансы).
4. **На что обратить внимание (риски)** — столкновения, наказания, активация Цзи Шэнь.
5. **2-3 конкретные рекомендации** — действия, направление, время.

### Правила при негативном прогнозе

- Никогда не пугать.
- Всегда давать конкретные рекомендации по минимизации.
- Напоминать о Триединстве (Небо + Земля + Человек) — карту мы не меняем, остальное в наших руках.
- Фокус на том, что человек МОЖЕТ сделать.

## Метафоры (из §6 базы)

- Такт удачи → «сезон»: зимой не суйся в прорубь, летом не сиди в шубе.
- Год удачи на хорошем такте → «попутный ветер на гонке»: можно выдать максимум.
- Столкновение года с натальной ветвью → «дорожный знак „объезд“»: путь не закрыт, но придётся менять маршрут.
- Юн Шэнь приходит → «попал в свою смену»: всё лёгкое.
- Цзи Шэнь приходит → «работаешь в ночную»: возможно, но устаёшь сильнее.

## Запреты

- Не называть конкретный день/время «свадьбы», «смерти», «увольнения».
- Не давать абсолютных «случится точно».
- При мрачных конфигурациях (тройные столкновения, наказание + Белый Тигр) — балансировать жёсткие факты конкретной защитой.
