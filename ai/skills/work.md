# Skill: Работа, карьера, деньги

При ответе на вопрос про работу/деньги используй механики ниже. Цитируй конкретные элементы из `[BAZI_DATA]` и `[CURRENT_MOMENT]` дословно.

## Что анализировать (порядок)

### 1. Элемент Богатства (財星)
- **正財 Прямое Богатство** — стабильный доход, зарплата, бережливость. Для мужчин — символ жены.
- **偏財 Косое Богатство** — бизнес, инвестиции, бонусы, удача. Для мужчин — символ любовницы/неформальной партнёрши.
- Есть ли Богатство в карте? Сильное или слабое?
- **Может ли Дневной Мастер «удержать» богатство?** Слабый Дневной Мастер + сильное Богатство = деньги приходят и уходят. Нужна сначала Печать/Братство.

### 2. Элемент Карьеры (官殺)
- **正官 Прямой Чиновник** — карьера в системе, найм, статус, дисциплина. Для женщин — символ мужа.
- **七殺 Семь Убийц** — предпринимательство, экстрим, конкуренция, амбиции. Для женщин — символ любовника.
- Баланс Чиновник vs Убийца: чистый Чжэн Гуань = карьера в найме; чистый Ци Ша при балансе (制殺) = огромная сила и лидерство; смешение = карьерные качели.

### 3. Самовыражение (食傷) как источник дохода
- 食神/傷官 порождают Богатство (cycle 我生 → 我克). Цепочка: талант → продукт → деньги.
- Сильное Самовыражение + Богатство = заработок на своих талантах.
- 傷官 без 印 = риск конфликтов с начальством; нужна 印 для смягчения.

### 4. Сферы деятельности через Полезное Божество (用神)

| Юн Шэнь | Подходящие сферы |
|---|---|
| Дерево | Образование, издательство, мода, текстиль, медицина (травы), мебель, ботаника |
| Огонь | IT, электроника, энергетика, маркетинг, медиа, ресторанный, красота, освещение |
| Земля | Недвижимость, строительство, сельское хозяйство, добыча, керамика, страхование, консалтинг |
| Металл | Финансы, банки, ювелирка, машиностроение, юриспруденция, правоохранение, хирургия |
| Вода | Логистика, транспорт, туризм, коммуникации, торговля, рыболовство, уборка, напитки |

### 5. Тактовый анализ (большая картина)
- В каком Такте Удачи (大運) клиент сейчас? Какой элемент?
- Приходят ли элементы Богатства/Карьеры в ближайшие 10 лет?
- Есть ли столкновения с финансовыми столпами (год/месяц)?
- Активирующий такт Богатства/Власти — окно для рывка.

### 6. Столп месяца — главный для карьеры
Столп месяца (16-32 года жизни, но в анализе — карьерная ось всегда):
- **Стиль работы:** Чем сильнее столп → тем доминантнее проявление в работе.
- **Структура карты (格局)** обычно определяется именно по сильнейшему Божеству в МЕСЯЧНОМ столпе. Это главный диагноз направления.

## Формат ответа по карьере

1. **Тип заработка** — найм / бизнес / фриланс / творчество. Через баланс Гуань-Ша + 財 + 食傷.
2. **Конкретные сферы** — через Полезное Божество (см. таблицу выше).
3. **Текущий период** — благоприятен или нет для крупных решений. Через текущий такт + год.
4. **2-3 конкретных шага** — что делать сейчас.

## Метафоры (из §6 базы)

- Сильное Богатство при слабом Дневной Мастер → «дырявое ведро»: вода льётся, не задерживается.
- Семь Убийц без контроля → «дикий конь»: или приручить, или сбросит.
- Шан Гуань в работе → «талантливый бунтарь»: один на один с начальством — тяжело, в своём проекте — горы свернёт.
- Структура Прямой Чиновник → «бегун с тренером»: правила = маршрут.

## Запреты

- Не давать конкретные инвестиционные советы («покупай X»).
- Не обещать конкретные суммы.
- Не предсказывать конкретный месяц увольнения/повышения — только диапазон + условие.
