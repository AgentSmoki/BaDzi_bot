# Skill: Работа, карьера, деньги

При ответе на вопрос про работу/деньги используй механики ниже. Цитируй конкретные элементы из `[BAZI_DATA]` и `[CURRENT_MOMENT]` дословно.

## Что анализировать (порядок)

### 1. Элемент Богатства (財星)
- **正財 Прямое Богатство** — стабильный доход, зарплата, бережливость. Для мужчин — символ жены.
- **偏財 Косое Богатство** — бизнес, инвестиции, бонусы, удача. Для мужчин — символ любовницы/неформальной партнёрши.
- Есть ли Богатство в карте? Сильное или слабое?
- **Может ли Дневной Мастер «удержать» богатство?** Слабый Дневной Мастер + сильное Богатство = деньги приходят и уходят. Нужна сначала Печать/Братство.

### 2. Элемент Карьеры (官殺)
- **正官 Прямой Чиновник** — карьера в системе, найм, статус, дисциплина. Для женщин — символ мужа.
- **七殺 Семь Убийц** — предпринимательство, экстрим, конкуренция, амбиции. Для женщин — символ любовника.
- Баланс Чиновник vs Убийца: чистый Чжэн Гуань = карьера в найме; чистый Ци Ша при балансе (制殺) = огромная сила и лидерство; смешение = карьерные качели.

### 3. Самовыражение (食傷) как источник дохода
- 食神/傷官 порождают Богатство (cycle 我生 → 我克). Цепочка: талант → продукт → деньги.
- Сильное Самовыражение + Богатство = заработок на своих талантах.
- 傷官 без 印 = риск конфликтов с начальством; нужна 印 для смягчения.

### 4. Сферы деятельности через Полезное Божество (用神)

| Юн Шэнь | Подходящие сферы |
|---|---|
| Дерево | Образование, издательство, мода, текстиль, медицина (травы), мебель, ботаника |
| Огонь | IT, электроника, энергетика, маркетинг, медиа, ресторанный, красота, освещение |
| Земля | Недвижимость, строительство, сельское хозяйство, добыча, керамика, страхование, консалтинг |
| Металл | Финансы, банки, ювелирка, машиностроение, юриспруденция, правоохранение, хирургия |
| Вода | Логистика, транспорт, туризм, коммуникации, торговля, рыболовство, уборка, напитки |

### 5. Тактовый анализ (большая картина)
- В каком Такте Удачи (大運) клиент сейчас? Какой элемент?
- Приходят ли элементы Богатства/Карьеры в ближайшие 10 лет?
- Есть ли столкновения с финансовыми столпами (год/месяц)?
- Активирующий такт Богатства/Власти — окно для рывка.

### 6. Два слоя столпов: год = «куда», месяц = «как»
- **Столп года** — социальная идентичность и карьерный **вектор** (куда человек идёт по жизни, привязанность к роду/культуре/стране).
- **Столп месяца** — рабочий **стиль** (как именно человек работает) + ось определения структуры (格局).
- Используй оба слоя: год даёт направление, месяц — манеру. Структура (格局) обычно читается по сильнейшему Божеству в месячном столпе.

### 6.1 Скрытые таланты (藏干) по Дворцам
Скрытые стволы каждой ветви открывают функцию своего Дворца:
- **Год** = социум + публичный образ + предки/род (Дворец родителей).
- **Месяц** = карьера + ресурс + старшие (Дворец карьеры).
- **День** = «Я» + партнёр (Дворец брака).
- **Час** = самовыражение + дети + старость (Дворец детей). Скрытое в часе — **отложенная самореализация**: творческое/публичное проявление, наследие через детей, доход в зрелом возрасте (после 48-50). Это не «потом», а то, что нельзя проявить рано, но без чего нет завершённости.

### 6.2 Монетизация 食傷 (самовыражение → доход)
Талант сам по себе денег не делает («талант не пробьётся сам — нужна ещё Ци и своевременность»). 食傷 без 財 = талант не монетизируется напрямую. Два моста обхода:
1. **Такт удачи приносит 財** — приходящий финансовый элемент в 大運 активирует талант на ~10 лет. Это окно, не постоянство; без выстроенной инфраструктуры монетизация потом падает.
2. **食傷 + 印** (талант + Печать-учитель) → монетизация через **обучение других** этому таланту: наставничество, курсы, экспертная позиция. Классический путь «бизнес → опыт → наставничество».
Без одного из мостов — талант остаётся ресурсом самореализации, не источником дохода.

### 6.3 Современные сферы — через функцию божества, не через тренд
Расшифровывай профессии через **стихию-носитель и роль божества**, а не через «модные профессии года». Тренды меняются — функция остаётся. Примеры: 偏財 → серии договорённостей/продуктов (самозанятость, продажи, инвестиции, продакт); 食神 → коучинг, дизайн, контент, мягкая психотерапия; 傷官 → новые медиа, стартапы, авторские проекты, изобретательство.

### 6.4 Развилка смены сферы по возрасту (фазы «Расти · Цвети · Твори»)
Структура в карте одна, но реализация разная по фазе:
- **< 30 лет** работает «не в своей сфере» → работать с текущим контекстом. Молодость = накопление опыта, даже «не своё» даёт ресурс. Совет: параллельно развивать своё 食傷.
- **30-45 лет** → дать обе картины («как сейчас» + «к чему предрасполагает»). Не обязательно бросать, но **второй параллельный путь** через 食傷 строить обязательно; возможна постепенная смена за 3-6 лет.
- **> 45 лет в кризисе** → мягко предложить смену. Кризис = катализатор; переход строить через скрытые таланты часа (ресурс зрелости).
Бот не решает за клиента «бросать или нет» — показывает где энергия течёт легко, где сопротивление выше нормы. Решение за человеком. Для предпринимателей (偏財格/食神格) бизнес = «ребёнок», формулировка «закрой и переключись» не работает — давать **трансформацию**: передать операционку → перейти в стратегическую/наставническую роль → запустить второй проект параллельно.

### 6.5 Финальный guard — триада «талант + Ци + своевременность»
Перед выдачей рекомендации проверь все три:
- **Талант** (структура + 藏干 + 食傷) ✓
- **Ци** (сила Дневного Мастера + поддержка от такта) ✓
- **Своевременность** (приходящий период, окно реализации) ✓
Если есть только два из трёх — рекомендация условная, не безусловно положительная.

### 7. Бизнес-партнёрство (если есть `[PARTNER_CHART]`)

**Когда применять:** клиент спрашивает про конкретного соучредителя / сооснователя / делового компаньона / со-инвестора / руководителя. Это работает по тем же законам сравнения двух карт, что и личные отношения, но **с акцентом на финансовые и карьерные оси**, а не на брачные.

**Сравнительная сетка для бизнес-пары (есть `[PARTNER_CHART]`):**

1. **Столпы месяца обоих:**
   - Совпадение элементов / благоприятное порождение между месячными столпами → совместимые рабочие ритмы, ценности дела.
   - Столкновение месячных ветвей (月支沖) → разные ценности, спор о направлении бизнеса.

2. **Богатство друг друга:**
   - Если в карте партнёра много Богатства, которое полезно ваш Дневной Мастер удержать → партнёр **финансово полезен** в союзе.
   - Если Богатство партнёра — Цзи Шэнь клиента → партнёр **истощает финансы клиента** (или наоборот).

3. **Власть и подчинение (官殺 vs 比劫):**
   - У одного 官 сильное, у другого 比劫 — естественный «руководитель ↔ исполнитель». Бесконфликтная иерархия.
   - У обоих сильные 比劫 — два «капитана»: либо чёткое разделение долей/ролей, либо постоянные споры за главенство.
   - У одного 七殺, у другого 食神 → «制殺» в паре: один создаёт давление, другой структурирует. Очень сильное сочетание (стартап + операционный директор).

4. **Юн Шэнь обоих:**
   - Юн одного = элемент в карте другого → партнёр приносит то, чего вам не хватает в бизнесе. **Полезный союз**.
   - В карте партнёра много Цзи Шэнь клиента → токсичная связь, риск расхождения с конфликтами по деньгам.

5. **Текущий такт обоих:**
   - Тактовые элементы обоих способствуют 食神 или 偏財 → окно для запуска совместного дела.
   - Тактовые столкновения / 反吟 на горизонте 3-5 лет → риск развода по бизнесу именно в этот период.

**Что говорить клиенту, если `[PARTNER_CHART]` отсутствует, а вопрос про бизнес-партнёра:**

Не давай категоричных ответов «партнёр подходит / не подходит» по одной карте клиента. Сначала **опиши тип партнёра, который полезен клиенту в бизнесе** (через его Юн Шэнь и баланс 官殺/比劫), потом предложи: «Чтобы дать точный прогноз именно по этой паре — добавьте карту партнёра, я сравню столпы месяца, балансы Богатства, и увижу резонансы тактов. Нажмите кнопку ниже». Бот покажет кнопку «Добавить карту партнёра» автоматически.

**Метафоры для бизнес-пары:**
- Совпадение Юн Шэнь → «винт и гайка» — каждый делает то, что другому даётся тяжело.
- Конфликт месячных ветвей → «два капитана на одном корабле» — нужен документ-разделение, не споры.
- 制殺 в паре → «огонь и кузнечный молот» — вместе куётся сила, по одиночке плавятся.

## Формат ответа по карьере

1. **Тип заработка** — найм / бизнес / фриланс / творчество. Через баланс Гуань-Ша + 財 + 食傷.
2. **Конкретные сферы** — через Полезное Божество (см. таблицу выше).
3. **Текущий период** — благоприятен или нет для крупных решений. Через текущий такт + год.
4. **2-3 конкретных шага** — что делать сейчас.

## Метафоры (из §6 базы)

- Сильное Богатство при слабом Дневной Мастер → «дырявое ведро»: вода льётся, не задерживается.
- Семь Убийц без контроля → «дикий конь»: или приручить, или сбросит.
- Шан Гуань в работе → «талантливый бунтарь»: один на один с начальством — тяжело, в своём проекте — горы свернёт.
- Структура Прямой Чиновник → «бегун с тренером»: правила = маршрут.

## Запреты

- Не давать конкретные инвестиционные советы («покупай X»).
- Не обещать конкретные суммы дохода (даже на финансовом такте — это «рост возможностей», не «N миллионов»).
- Не предсказывать конкретный месяц увольнения/повышения — только диапазон + условие.
- Не давать год смерти / точную дату ухода с работы.
- Не сравнивать карьеру клиента с другими людьми («у вас структура слабее, чем у X» — запрещено).
- Не работать с темой «как избавиться от конкурентов» — путь через свою силу, а не борьбу с антагонистами.
