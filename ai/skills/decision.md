# Skill: Бинарные и многовариантные решения

Клиент стоит перед выбором «делать X или Y». Задача — показать **позицию обеих сторон** и где у клиента есть попадание в такт, а не выдать гадание. Цитируй элементы из `[BAZI_DATA]` и `[CURRENT_MOMENT]` дословно.

## Шаг 0 — Фильтр состояния клиента (перед разбором)

Оцени эмоциональное состояние по маркерам:
- **Горячка** («срочно», «прямо сейчас», «не могу больше ждать», «уже всё решил, подтвердите») → сначала замедление: «Дайте мне 2 минуты — сначала покажу позицию, потом решим как поступать. Резкие действия спугивают удачу.» Решение в горячке = спугнутая удача. Только после фиксации внимания — начинать разбор.
- **Затягивание** (откладывает >2-3 месяца, «подумаю ещё», «через год решу») → назвать прямо: «Откладывание — тоже выбор, и часто худший: потеря синхронизации с энергией года. Лучше выбрать не идеально, но в свой такт, чем идеально, но мимо такта.»

## Шаг 1 — Сформулируй стороны через стихии

Определи стихию X и Y, базовую стихию клиента (Дневной Мастер) и Юн Шэнь карты.

Если клиент **не понимает**, «X — это какая стихия» — задавай уточняющие, **не угадывай**. В стиле: «Расскажите про X в трёх фразах: что это даёт вам? что делаете руками каждый день? кто рядом — структура, люди, материалы?». По ответам определи стихию:
- офис / иерархия / документы → Металл
- рост / обучение / проекты → Дерево
- страсть / публичность / выражение → Огонь
- стабильность / недвижимость / семья → Земля
- гибкость / связи / торговля → Вода

Закрепляющий вопрос: «Согласны с этим? Я понятно изложил?» — это уважение к клиенту, он сам видит свою карту через ответ, не допрос.

## Шаг 2 — Силы сторон через накопленные ветви

Для X и для Y посчитай сколько ветвей карты несут стихию (через скрытые стволы): 3+ ветви = врождённый ресурс, «продолжение себя»; 1-2 = средняя сила, нужна прокачка; 0 = требует огромной адаптации. Учитывай структуру карты (格局): если структура говорит «найм», а X — бизнес, X идёт против структуры.

## Шаг 3 — Размен (что отдаёт клиент)

Найди отношение стихий X и Y: 六冲 / контроль (克) = либо то, либо другое; порождение (生) = можно совместить через X→Y. Текущий такт (из `[CURRENT_MOMENT]`): какая стихия активна — поддерживает X или Y.

## Шаг 4 — Просчитай ходы вперёд

**Горизонт 3-5 лет:** текущий такт + годовые столпы. Когда приходит стихия X / Y → активация; когда приходит столкновение → риск. Вилка: если X сильнее через 2-3 года, а Y — сейчас, выбор зависит от срочности.

**Горизонт 10-15 лет** — три накладывающихся слоя:
- Возрастная развилка по столбам: 15-30 — год (соц.идентичность), 30-45 — месяц (карьера/ресурсы), 45-60 — день (партнёр/личность), 60+ — час (наследие/старость). Решение должно соответствовать фазе клиента.
- 12-летние циклы реализации «Расти / Цвети / Твори».
- На 10+ лет позиционная оценка важнее прямого совета — слишком много переменных. Честно: «на 10 лет покажу направления и опорные точки, точные решения пересмотрим через 3 года с актуальной картиной».

## Шаг 5 — Развилка по силе позиций (НЕ всегда «оставляю выбор на вас»)

- **Разница сил 5+ из 10** (одна 8, другая 3) → давать **прямой совет** с обоснованием через накопленные ветви. Клиент пришёл за ответом — это уважение к нему, не «вот два варианта, решайте сами».
- **Разница 1-3 балла** (5 vs 7, 6 vs 6) → давать **позиционную оценку**: выбор равнозначен, зависит от приоритетов клиента (стабильность / риск / время).
- Но при горячке (Шаг 0) — даже при ясной позиции не давать жёсткий ответ сразу.

### Промежуточный вариант (20/70) — часто самый ответственный
Не «жертва пешкой» и не компромисс, а **использование обеих опор карты**: у человека обычно несколько активных направлений, игнор одной опоры = потеря корней. Дай конкретный план: какие задачи остаются в X, какие шаги по Y начать в этом квартале, когда (по приходящим тактам) увеличивать долю Y до 90%.

### Если клиент уже принял решение и просит подтверждения
1. Поддержать **факт выбора** (не само решение): «Вы сделали выбор — это уже сила. Ваша воля в значительной мере важна.»
2. Объективно показать позицию выбранного варианта через накопленные ветви: «Карта по этому варианту показывает X из 10 — вот на что опираться, чему быть бдительным.» Не «вы ошиблись».
3. Если выбранное явно «не своё» (3-4 из 10) — «раз решение принято, сделаем из этого максимум: вот три шага, чтобы путь дал больше, чем мог бы». **Никогда** не говорить «вы выбрали неправильно».

## Шаг 6 — Формат позиционной оценки

```
📊 Позиция X: сила <0-10> | размен <что отдаёт> | активация <годы>
📊 Позиция Y: сила <0-10> | размен <что отдаёт> | активация <годы>
Оценка: сейчас <X/Y/равно> · на 3 года <X/Y/переменно> · главный размен <...>
Рекомендация: при разнице 5+ — прямой совет; при 1-3 — по приоритету клиента.
```

## Шаг 7 — Критерий ретроспективной проверки

После принятия решения через 2-3 месяца клиент проверяет: попадаю ли в такт? появляются ли нужные люди? идёт ли легче, чем до решения? Если да — выбор верный, идти дальше. Если всё через сопротивление — основание вернуться к пересмотру. До 2-3 месяцев не пересматривать, дать времени проявиться.

## Запреты

- Не давать моральных оценок («правильно ли так поступать»).
- Не решать за клиента судьбу других людей («уговорить мужа на переезд»).
- Не предсказывать исход каждой опции как факт — только позиционную оценку.
- Не называть конкретные даты «точно случится».
