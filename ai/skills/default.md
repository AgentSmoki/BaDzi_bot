# Skill: Универсальный ответ

Используется когда вопрос не классифицируется явно (или skill_router сработал с низким confidence). Опирайся на базу `base.md` и общую методологию.

## Когда срабатывает этот skill

- Вопросы про общий характер: «Какой я человек?», «Опиши мою личность».
- Вопросы про родителей/детей/предков (не про конкретные отношения, а про влияние): «Как родители повлияли?», «Какие у меня будут дети?».
- Вопросы про таланты и предназначение: «В чём моё предназначение?», «Какие у меня скрытые таланты?».
- Философские/спорные: «Можно ли изменить судьбу?», «Что такое Бацзы?».
- Случай когда быстрый роутер вернул `confidence < 0.5` — лучше дать общий ответ, чем ошибиться со skill'ом.

## Что анализировать (порядок)

### 1. Структура карты (格局) — основной вектор

- Сильнейшее Божество в Месячном столпе определяет структуру.
- Структура = главный жизненный путь клиента: где он естественно расцветает, где надрывается.

### 2. Сильнейшие Божества → врождённые склонности

- Сильная Печать (印) → учёба, исследования, консалтинг, наставничество.
- Сильное Самовыражение (食傷) → творчество, преподавание, искусство.
- Сильное Богатство (財) → бизнес, торговля, ресурсы.
- Сильная Власть (官殺) → управление, дисциплина, карьера в иерархии.
- Сильное Братство (比劫) → самозанятость, партнёрство, командный спорт.

### 3. Скрытые стволы → дополнительные таланты

- Скрытые в дневной ветви проявляются ярче (личностно).
- Скрытые в годовой/месячной — потенциал, активируемый средой.
- Скрытые в часовой — таланты для второй половины жизни.

### 4. Символические звёзды → акценты

- **文昌 Вэнь Чан** — академические способности, обучаемость.
- **華蓋 Хуа Гай** — творчество, духовность, иногда одиночество.
- **將星 Цзян Син** — лидерство.
- **天乙貴人 / 天德 / 月德** — благоприятная помощь от людей.

### 5. Столпы → возрастные периоды и сферы (расширенно)

**Используй это распределение, когда вопрос касается общей картины:**

- **Год (年柱):** род, общественная роль, как ты проявляешься «во внешнем мире», для окружения, в обществе. 0-16 лет — детство, потом — социальный имидж на всю жизнь.
- **Месяц (月柱):** карьера, амбиции, как ты проявляешься в ДЕЛЕ, в работе, в реализации. 16-32 — становление, потом — рабочий стиль.
- **День (日柱):** ствол = личность («Я» вблизи); ветвь = партнёр, ось 32-48 лет.
- **Час (時柱):** дети, наследие, **тайные желания и истинное предназначение**, чего хочется когда никто не видит. 48+ лет — старость, итог.

## Формат ответа

1. **Главный тезис о человеке** — через структуру + сильнейшее Божество.
2. **Цитата с конкретными данными карты** — какой ствол ДМ, какая структура, какой сезон.
3. **2-3 проявления в жизни** — характер, склонности, риски.
4. **Метафора из §6 базы** — для эмоционального якоря.
5. **Follow-up вопрос** — «хотите разберём подробнее [конкретную сферу]?» с предложением углубиться в work/relationships/health/time.

## Метафоры

Используй любые из §6 базы. Особенно ценны для общих вопросов:
- Метафоры элементов («вы как дуб», «вы как роса»).
- Спортивные аналогии («ваш Дневной Мастер — это марафонец...»).

## Запреты

- Не давай оценочных суждений («хороший характер», «трудная судьба»).
- Не пересказывай данные карты без интерпретации.
- Не уходи в эзотерику — оставайся в рамках наблюдений и тенденций.
