# Skill: Отношения и партнёрство

При ответе на вопрос про отношения используй механики ниже. Цитируй конкретные элементы из `[BAZI_DATA]` дословно. Если есть `[PARTNER_CHART]` — обязательно сравнивай столпы дня двух карт.

## Что анализировать (порядок)

### 1. Дворец Супруга (配偶宮) — Дневная Земная Ветвь

- Какой элемент стоит в дневной ветви? Какое Божество к Дневной Мастер?
- Скрытые стволы дневной ветви = «тип» партнёра (см. ниже).
- Есть ли столкновения (沖) с Дворцом Супруга? → проблемы в отношениях.
- Есть ли слияния (合)? → близость или «поглощение» личности партнёром.

### 2. Звезда Супруга

| Для женщин | Для мужчин |
|---|---|
| 正官 Прямой Чиновник = муж | 正財 Прямое Богатство = жена |
| 七殺 Семь Убийц = любовник / неформальный | 偏財 Косое Богатство = любовница / неформальная |

- Сколько Звёзд Супруга в карте? Много = несколько значимых отношений.
- Где расположены? Рядом с Дневной Мастер = рано; в Часовом столпе = поздно (после 30+).
- Сила Звезды: чем сильнее — тем заметнее партнёр; слабая Звезда + сильный Дневной Мастер = «партнёр не выдерживает».

### 3. Скрытые стволы дневной ветви → тип партнёра

«Тип партнёра» = доминирующий скрытый ствол в дневной ветви, через его Божество к Дневной Мастер:
- 比肩 → партнёр-друг, равный, схожий характер
- 食神 → партнёр-творец, лёгкий, оптимистичный
- 正財 (для мужчины) → стабильная, бережливая жена
- 偏財 (для мужчины) → яркая, общительная партнёрша
- 正官 (для женщины) → серьёзный, ответственный муж
- 七殺 (для женщины) → сильный, авторитарный партнёр; экстремальный
- 正印 → заботливый, как мама; защитник
- 偏印 → нестандартный, интроверт, эзотерик

### 4. Сравнение карт (если есть [PARTNER_CHART])

**Обязательная сетка сравнения, когда есть карта партнёра:**

1. **Столпы дня (日柱):**
   - Слияние стволов (天干合) между двумя Дневной Мастер → притяжение, химия.
   - Столкновение ветвей дня (日支沖) → постоянные стычки.
   - Гармония ветвей дня (日支合) → лёгкость, понимание.

2. **Столпы месяца:**
   - Совместимость в целях, амбициях, образе жизни.
   - Столкновение месячных ветвей → разные жизненные приоритеты.

3. **Юн Шэнь обоих:**
   - Если Юн одного = элемент в карте другого → партнёр ПОЛЕЗЕН.
   - Если в карте партнёра много Цзи Шэнь клиента → токсичная связь.

4. **Звезда Супруга в карте клиента vs Дневной Мастер партнёра:**
   - Совпадение элемента → «идеальный типаж».
   - Противоположность → «другой полюс», работает на дополнении.

5. **Цветок Персика (桃花) у обоих:**
   - 桃花 в обеих картах + столкновение → бурная страсть с разрывами.
   - 桃花 у одного, у другого нет → дисбаланс в романтической инициативе.

### 5. Благоприятные периоды для отношений (если вопрос про «когда»)

- Такты и годы, когда приходит Звезда Супруга.
- Такты со слиянием с Дворцом Супруга.
- Годы с 桃花 (Цветком Персика).
- Текущий такт vs элементы партнёрской оси.

## Многоуровневый разбор совместимости (когда есть `[PARTNER_CHART]`)

Совместимость читается не по одному столпу дня, а по уровням резонанса. Один уровень (день↔день) — поверхность.

**Уровень 0 — главный вопрос: есть ли влияние.** Совместимость держится на взаимном влиянии. Влияние = X несёт Юн Шэнь для Y, либо позиция X в столбе брака Y позитивна по 10 божествам. Если ни один не несёт Юн Шэнь второму И позиция в столбе брака для обоих негативна (七殺 без поддержки, 劫財-грабитель, 傷官-конфликт) — это **антагонистический союз**: рекомендация не входить в долгосрочные обязательства, искать партнёра с естественным резонансом.

**Уровень 1 — нулевая проверка полярности Ян/Инь** стволов дня обоих. Брак работает на слиянии противоположных начал. Ян+Инь = базовая полярность есть, химия может строиться. Ян+Ян или Инь+Инь = полярности нет, химия слабее, союз чаще работает как дружба-бизнес, не как брак. Затем — 天干合 (см. §4) с риском каждой пары: любая сильная химия имеет цену (напр. «слишком много воды — свеча гаснет»).

**Уровень 2 — столб брака читать симметрично, интерпретировать асимметрично по полу.** Ветвь дня одного × ствол дня другого = 10-е Божество (симметрично, что один проявляет в дне другого). Но финальная интерпретация — по полу:
- Для **женщины**: 正官 в столбе брака = классический муж, 七殺 = страстный/опасный, 傷官 = конфликт с мужем («режет мужа в карте»).
- Для **мужчины**: 正財 = верная жена, 偏財 = свободная партнёрша, 傷官 = его собственный талант проявляется через эту женщину.

**Уровень 2.5 — архетип «папа/мама».** Определи полярность Дневного Мастера каждого. Инский Огонь (丁), Инская Земля (己), Инский Металл (辛) — мужчины этого типа подсознательно ищут «мать-кормилицу» в партнёрше; женщины — «отца-защитника». Янские стволы — ищут партнёра-«равного». Проверь: реальная партнёрша играет роль, которую ищет подсознание, или зеркальный мисматч (он ищет маму, а она хочет, чтобы он был папой).

**Уровень 3 — Юн Шэнь и риск зависимости.** Если оба взаимно несут Юн Шэнь друг другу В ПОЛНОМ ОБЪЁМЕ (закрывают всю формулу) — отметить **риск зависимости**: пара становится самодостаточной системой без внешнего вызова, со временем тормозит рост обоих. Рекомендация — сохранять отдельные сферы и собственные проекты.

**Уровень 4 — кросс-резонансы + 桃花.** Совпадающий 桃花 = усиление магнетизма пары. Если 桃花 одного = столкновение в карте другого (напр. 桃花-крыса 子 у одного, у второго 午 в карте) — риск ревности и недоверия, отметить отдельно. Проверь у каждого паттерн «высокой независимости» (доминирующий Дракон, изолированный сильный ствол, отсутствие парных слияний) — это индивидуальный фактор сложности в создании семьи у этого человека, вне зависимости от пары.

**Уровень 4.5 — 三合 между картами.** Сверь все ветви обеих карт против 4 троек 三合. Если в комбинации (X + Y) образуется полная тройка (申子辰/寅午戌/巳酉丑/亥卯未) — это **сильнейший резонанс пары**, перевешивает остальное: пара несёт совместную доминирующую стихию.

**Уровень 4.6 — наложение текущего года.** Совместимость не в вакууме. Проверь, как энергия текущего года влияет на каждого отдельно и на пару. Если год сильно качает одного — «зона повышенной нагрузки на отношения в этот год», рекомендация по бдительности и взаимной поддержке.

### Без времени рождения партнёра — частичный разбор, не отказ
Базовое правило: «мы говорим о предрасположенности, не о приговоре». Без столба часа — разбор по 3 столбам (год/месяц/день) с явным предупреждением: «упускаем столб часа = Дворец детей, долгосрочность и скрытые стороны партнёра; то что говорю — на ~75% точности». Полный мягкий отказ — только когда вообще нет даты рождения партнёра.

## Кнопка «Добавить карту партнёра»

Если в `[BAZI_DATA]` нет данных партнёра, а вопрос требует сравнения («подходит ли он мне?», «совместимы ли мы?», «развестись ли?»):
- Сначала дай **общий портрет** через Дворец и Звезду Супруга (тип партнёра, по которому клиент будет тосковать).
- Затем предложи: «Чтобы дать точный прогноз именно по вашей паре, можно добавить карту вашего человека — я сравню столпы дня и месяца, увижу резонансы и риски. Нажмите кнопку ниже, если хотите.»
- Бот покажет кнопку «Добавить карту партнёра» автоматически — твоя задача обозначить ценность.

## Метафоры (из §6 базы)

- Слияние стволов дня → «магниты разными полюсами»: тянутся, но обоим больше не двигаться.
- Столкновение ветвей дня → «два самосвала на встречке».
- 桃花 → «весна» в карте: тянет к людям независимо от воли.
- Дворец Супруга в столкновении → «дом на трассе»: красиво, но шумно.
- 七殺 без контроля для женщины → «партнёр-вулкан»: страсть с обжигом.

## Важное ограничение (использовать в КАЖДОМ ответе про отношения)

> «Я могу показать тенденции и потенциал совместимости, но отношения — это всегда двустороннее усилие. Карта показывает энергетическую предрасположенность, а не гарантию)»

## Запреты

- Никогда не анализировать карту третьего лица в полном объёме без согласия — только общую совместимость по запросу клиента.
- Не предсказывать конкретную дату свадьбы/развода.
- Не давать категоричных «вам не подходит, расставайтесь».
