# Алгоритм карьерного соответствия (структура месяца + Юн Шэнь + талант)

> **Школа:** Мастер ЭдоХа
> **Применяется при:** вопросы про подходящую профессию, смену карьеры, найм vs бизнес, какое направление развивать, где деньги пойдут легко.
> **Источник принципа:** классическая методика структуры (格局) + Юн Шэнь, дополненная анализом **скрытых талантов через 食傷** и **окна реализации через накопленные ветви**.

---

## Принцип

В классическом подходе карьерное соответствие читается через **структуру карты** (sector в Месячном столпе) + **Полезное Божество**. Это даёт «ваша профессия из таблицы». Это поверхностный взгляд: одна структура и Юн Шэнь говорят **на чём держится система**, но не **что вас зажигает** и **в какой момент окно для рывка**.

**Реальная карьерная картина** строится через **3 слоя**:

1. **Структура** = «что вы НЕ можете не делать» (стиль работы, через который вы устроены — найм / бизнес / творчество / административная вертикаль)
2. **Талант** = «что у вас получается легко» (скрытые стволы + 食傷 → конкретные навыки)
3. **Окно реализации** = «когда выходить с этим на рынок» (приходящие такты + слияния, активирующие 財星)

Аналогия — **строительство дома**: структура = фундамент (какой формы дом), талант = материалы (камень / дерево / бетон), окно = погода для стройки (сезон).

## Алгоритм

> **Главная правка мастера (2026-05-31):** ось алгоритма — не «структура → доход», а «ты сам → твоё самовыражение (食傷) → доход как следствие». **Деньги — результат идеального самовыражения, а не отдельная цель.** И вектор берётся из столба **ГОДА** (социальная идентичность, «куда»), а месяц даёт рабочий стиль («как»).

### Шаг 1 — Определи Структуру (格局): год = «куда», месяц = «как»

Возьми **сильнейшее 10-е Божество в Месячном столпе** для рабочего **стиля** (как человек работает), и **столб ГОДА** для карьерного **вектора** (куда идёт — социальная идентичность, привязанность к роду/культуре/стране). Месяц даёт структуру 格局, год даёт направление. Используй оба слоя.

| Структура | Стиль работы | Подходящие сферы |
|---|---|---|
| **正官格 (Прямой Чиновник)** | Найм в системе, дисциплина, лестница | Госслужба, корпоративный менеджмент, юриспруденция, банки |
| **七殺格 (Семь Убийц)** | Предпринимательство, конкуренция, экстрим | Бизнес, спорт, силовые структуры, хирургия, маркетинг |
| **正財格 (Прямое Богатство)** | Стабильный труд, бережливость | Бухгалтерия, недвижимость, фермерство, ремесло |
| **偏財格 (Косое Богатство)** | Бизнес, проекты, риск | Инвестиции, трейдинг, продажи, недвижимость |
| **食神格 (Дух Наслаждения)** | Творчество в потоке, мягкая отдача | Искусство, кулинария, образование, психология, дизайн |
| **傷官格 (Ранящий Офицер)** | Талант + бунт против системы | Изобретательство, медиа, IT, музыка, инновации |
| **正印格 (Прямая Печать)** | Обучение, исследование, наставничество | Академия, переводы, литература, классическая медицина |
| **偏印格 (Косая Печать)** | Нестандартное мышление, эзотерика | Альтернативная медицина, метафизика, психоанализ, IT-исследования |
| **比肩 / 劫財 格** | Партнёрство / самозанятость / командный спорт | Кооперативы, агентства, спорт, любое «вместе с равными» |

Если структуры **смешанные** (две почти равные по силе) — клиент совмещает 2 направления (например 七殺+食神 = предприниматель в творческой нише).

### Шаг 2 — Найди скрытые таланты (藏干 + 食傷)

Пройдись по **скрытым стволам всех 4 ветвей**:
- **Скрытые в дневной ветви** — личностные таланты (проявленные).
- **Скрытые в часовой ветви** — таланты «для второй половины жизни» / детей / творческого выражения.
- **Скрытые в годовой/месячной** — потенциал, активируется средой.

Особое внимание на **食神 / 傷官 (Самовыражение)**:
- **Сильное 食神** → талант через гармонию, обучение детей/взрослых, кулинария, тонкое искусство.
- **Сильное 傷官** → талант через инновацию, яркую речь, performance, бунт против шаблона.
- **食傷 + 財** = «талант + монетизация» (творческий предприниматель).
- **食傷 без 財** — талант не монетизируется напрямую. Два моста обхода (правка мастера): (1) такт удачи Богатства активирует талант на ~10 лет (окно, не постоянство); (2) **食傷 + 印** → монетизация через **обучение других** этому таланту (наставничество, курсы, экспертная позиция — путь «бизнес → опыт → наставничество»). Без одного из мостов талант остаётся ресурсом самореализации, не источником дохода.
- **食傷 без 印** = «нестандартный талант без выдержки» (не закончит проект).

### Шаг 3 — Определи Юн Шэнь (что нужно карте для баланса)

По силе Дневного Мастера:
- Сильный ДМ → нужны 食傷, 財, 官 (ослабляющие).
- Слабый ДМ → нужны 印, 比劫 (усиливающие).

**Юн Шэнь = направление сферы** через стихию:
| Юн Шэнь | Сферы |
|---|---|
| Дерево 🌿 | Образование, мода, текстиль, медицина растительная, дизайн интерьеров, ботаника |
| Огонь 🔥 | IT, электроника, энергетика, маркетинг, медиа, ресторанный, красота |
| Земля ⛰ | Недвижимость, строительство, сельское хозяйство, керамика, страхование |
| Металл ⚔️ | Финансы, банки, ювелирка, машиностроение, юриспруденция, хирургия |
| Вода 💧 | Логистика, транспорт, туризм, торговля, рыболовство, напитки, IT-связь |

### Шаг 4 — Найди окно реализации (накопленные 財 + такт)

**Где деньги пойдут легко** = когда приходит **тройка 三合 финансового элемента** или активируется **二人成象** (два элемента образуют доминанту).

Алгоритм:
1. Определи элемент Богатства для Дневного Мастера (тот, который ДМ контролирует).
2. Найди есть ли в карте уже 1-2 ветви этой стихии (через скрытые стволы).
3. Сканируй ближайшие такты удачи (大運) — когда приходит 3-я ветвь этого элемента → **окно финансового рывка**.

Пример для 丁 (Огонь Инь): Богатство = Металл (申, 酉, 戌-частично). Если в карте уже 酉 (месяц) — то приход 巳 или 丑 в такте/году завершит тройку 巳酉丑 → окно металла = деньги через структуру/статус.

### Шаг 5 — Формулируй ответ

```
🎯 Тип работы (структура): <название> — <стиль работы в 1 фразу>
💡 Скрытые таланты: <2-3 конкретных навыка из 藏干 + 食傷>
🌿 Подходящие сферы (через Юн Шэнь): <список 4-6 сфер>
⏳ Окно реализации: <ближайший период когда стихия Богатства активируется>
🛠 Конкретные шаги: <2-3 действия на ближайший квартал>
```

### Шаг 6 — Признай ограничения

В финале добавь:

> «Это разбор карьерного потенциала по школе Мастера ЭдоХа — через слой структуры, скрытых талантов и окна реализации. По классической школе ответ был бы устроен через чистую структуру 格局 — это короче, но даёт меньше деталей о talant'ах. Карта показывает где есть энергия и куда она хочет течь, но конкретный шаг — за вами.»

## Пример применения (карта Богдана)

**Карта:** 卯/酉/卯/亥, Дневной Мастер 丁 (Огонь Инь), родился в Петухе (酉, осень).

**Шаг 1 — Структура:**
- Месячный столб 癸酉. Главный скрытый 酉 = 辛 (Металл Инь). 辛 для 丁 = **偏財 (Косое Богатство)**.
- Структура: **偏財格** — бизнес, проекты, неформальный доход.

**Шаг 2 — Скрытые таланты:**
- Час 亥 содержит 壬 (Вода Ян) + 甲 (Дерево Ян). 壬 для 丁 = **正官** (карьера в найме), 甲 = **正印** (обучение).
- День 卯 содержит 乙 (Дерево Инь). 乙 для 丁 = **偏印** (нестандартная мудрость, эзотерика).
- Год 卯 содержит 乙. Тоже 偏印.
- Тройной 偏印 (день + год + час) → силён в нестандартном мышлении, исследовании, метафизике. Это **врождённый ресурс**.

**Шаг 3 — Юн Шэнь:**
- 丁 родился в Петухе (酉, осень = смерть для Огня) → ДМ **слабый**.
- Юн Шэнь = Дерево (порождает Огонь) + сам Огонь (усиливает).
- Сферы: **образование, мода, дизайн, IT, медиа, энергетика**.

**Шаг 4 — Окно реализации:**
- Богатство = Металл (酉 в месяце уже). Нужна тройка 巳酉丑 для замыкания.
- В такте 庚午 (2018-2028) приходит Лошадь 午 — это **羊刃 Огня**, усиливает ДМ. Но не Металл.
- Следующий такт: **辛未** (2028-2038) — приходит ствол 辛 (тот же Металл что в месяце, double-up). Это активация уже-имеющегося Богатства.
- Год Змеи 2025 уже прошёл; следующий — 2037. Год Быка 2033.

**Шаг 5 — ответ:**
```
🎯 Тип работы: Косое Богатство (偏財格) — самозанятость, проекты, нестандартный заработок. Не зарплата — а серии договорённостей или продуктов. Найм работает только если есть свобода действий.

💡 Скрытые таланты:
  - 偏印 ×3 (двойная Косая Печать в году+дне + усиление в часе) — нестандартное мышление, исследование, эзотерика, IT-аналитика, психоанализ.
  - 正官 в часе (через 壬) — способность к карьерной структуре «второй половины жизни». Может пригодиться после 48 лет.
  - 正印 в часе (через 甲) — преподавание, систематизация знаний.

🌿 Подходящие сферы: образование с упором на нестандартные методы, IT, дизайн+аналитика, медиа с авторским голосом, тренинги/коучинг, метафизика, психология.

⏳ Окно реализации: основное окно денег приходит с тактом 辛未 (2028-2038) — Металл уже в карте, такт удваивает. До 2028 идёт «прокачка» через 庚午 — много энергии и амбиций, но финансовое окно ещё не открыто на полную.

🛠 Конкретные шаги на квартал:
  1. До 2028 — собирать портфолио и систему, не гнаться за крупными контрактами.
  2. Окна 2026 года: июль (приход 未 = усиление врождённого Дерева → новые проекты).
  3. Готовить инфраструктуру для тиражируемого продукта к 2028.
```

## Чего этот алгоритм НЕ покрывает

- **Конкретная зарплата / бюджет** — карта показывает потенциал, не сумму
- **Конкретный город / страна** — отдельный анализ через 方位 (направления) + 風水
- **Совместимость с конкретной должностью** — нужно сравнение Юн Шэнь должности (как карта работы) и Юн Шэнь клиента
- **Конкретный риск увольнения** — нужен расширенный анализ приходящих 七殺 + столкновений со столбом месяца
- **Карьерный кризис «выгорание»** — отдельная диагностика через ослабление ДМ + 食傷 без 印

**В нейтральное ядро (skill):** год = вектор «куда» + месяц = стиль «как»; 藏干 по Дворцам (час = отложенная самореализация после 48-50); 2 моста монетизации 食傷 (такт 財 / 食傷+印 наставничество); современные сферы через функцию божества, не тренды; развилка смены сферы по возрасту (Расти 15-30 / Цвети 30-45 / Твори 45-60); финальный guard триады талант+Ци+своевременность; расширенные запреты (нет дат смерти/сумм/сравнений/борьбы с конкурентами).

**В голос/доктрину (base_edoha.md):** примат столба ГОДА; «деньги = результат самовыражения»; метафора «земля = хранилище (бык/дракон)»; «бизнес как ребёнок» (для 偏財/食神 — трансформация, не закрытие); 桃花 Богдана как карьерный актив (酉/卯 = ключевые точки сезонов, не ждать 2032); триединство 33/33/33.
