# Алгоритм оценки благоприятных периодов (三合 + 桃花 + 貴人)

> **Школа:** Мастер ЭдоХа
> **Применяется при:** вопросы про благоприятные периоды, время для рывка, окна возможностей, удачные месяцы для крупных решений (свадьба, переезд, новый проект, инвестиции).
> **Источник принципа:** транскрипция консультации сенсея 2026-05-21 + классические правила 三合 / 桃花 / 天乙貴人, переосмысленные через накопленные силы.

---

## Принцип

В классическом подходе благоприятным считается любой период, когда приходит **Полезное Божество** (用神) или **благородная звезда** (貴人). Это поверхностный взгляд: одна звезда сама по себе создаёт **возможность**, но не **окно**.

**Реальное окно возможностей** возникает когда приходящий период активирует **три-стороннее слияние** (三合) с двумя ветвями карты рождения, образуя полный треугольник одной стихии. Это создаёт паттерн **3-out-of-3** — энергия не просто присутствует, она замыкается в самоподдерживающийся цикл, как трое музыкантов держат темп друг для друга.

Военная аналогия здесь не подходит — это не битва. Подходит метафора **трёх друзей**: один человек поднимает идею, второй её углубляет, третий доводит до результата. Каждый сам по себе не справится, но вместе — переворачивают ситуацию.

---

## Алгоритм

### Шаг 1 — Найди потенциальные тройки слияния (三合) в карте рождения

Пройдись по 4 ветвям карты рождения (год / месяц / день / час). Сгруппируй те, что входят в одну из 4 классических троек:

| Тройка 三合 | Стихия | Ветви |
|---|---|---|
| 申子辰 (申-Обезьяна, 子-Крыса, 辰-Дракон) | Вода | Богатство, мудрость, поток |
| 寅午戌 (寅-Тигр, 午-Лошадь, 戌-Собака) | Огонь | Слава, проявленность, страсть |
| 巳酉丑 (巳-Змея, 酉-Петух, 丑-Бык) | Металл | Структура, статус, дисциплина |
| 亥卯未 (亥-Свинья, 卯-Кролик, 未-Коза) | Дерево | Рост, гибкость, проекты |

Категории:
- **2 ветви тройки уже в карте рождения** = «двое друзей ждут третьего». Это **зона потенциального окна возможностей**.
- **1 ветвь тройки в карте** = одиночка, окно слабое.
- **3 ветви тройки в карте** (редко) = карта изначально несёт эту стихию как **врождённый ресурс** — но без приходящей активации он спит.

### Шаг 2 — Сканируй приходящие годы и месяцы (горизонт ±2 года)

Для каждого приходящего года (流年) и месяца (流月) определи годовую/месячную ветвь. Сверь с потенциальными тройками из Шага 1.

### Шаг 3 — Применяй паттерны открытия окна (от самого сильного к слабому)

#### Большое окно: полная тройка 三合 замыкается
2 ветви тройки в карте + 3-я приходит → **полный треугольник** активируется. Стихия становится доминирующей на этот период.

**Пример:** в карте 卯 (год) и 未 (час) — две из тройки Дерева 亥卯未. Когда приходит 亥 (год Свиньи или месяц ноябрь) → треугольник Дерева замыкается → окно для роста, новых проектов, гибких решений длиной 1-3 месяца.

#### Среднее окно: полу-слияние (半合)
В карте есть пара из соседних элементов тройки + приходит третья:
- 申子 / 子辰 → полу-Вода (без полной тройки)
- 寅午 / 午戌 → полу-Огонь
- 巳酉 / 酉丑 → полу-Металл
- 亥卯 / 卯未 → полу-Дерево

Это **средняя сила** — окно открывается, но не на 100%. Подходит для подготовки, не для рывка.

#### Точечная возможность: 桃花 (Персиковый Цветок)
Приходящий год или месяц приносит **桃花** (子, 午, 卯, 酉) — точечная активация магнетизма, отношений, творчества. Сам по себе **не окно**, но усиливает любое уже открытое окно.

Правило выбора 桃花:
| Если в карте день/год | 桃花 = |
|---|---|
| 申/子/辰 | 酉 |
| 寅/午/戌 | 卯 |
| 巳/酉/丑 | 午 |
| 亥/卯/未 | 子 |

#### Защита и сопровождение: 天乙貴人 (Небесный Благородный)
Приходящие столпы со стволами, дающими 天乙貴人 для вашего Дневного Мастера — **охранник окна**. Не открывают окно сами, но снижают риски в нём на 50-70%.

| Дневной Мастер | 天乙貴人 ветви |
|---|---|
| 甲/戊/庚 | 丑, 未 |
| 乙/己 | 子, 申 |
| 丙/丁 | 亥, 酉 |
| 壬/癸 | 卯, 巳 |
| 辛 | 午, 寅 |

### Шаг 4 — Ранжируй окна и формулируй ответ

Дай **топ-3** окон в порядке убывания силы:

```
🥇 Большое окно: <месяц/год X> — полная тройка 三合 <стихия>: <ветви> + 桃花/貴人 (если есть)
🥈 Среднее окно: <месяц/год Y> — полу-слияние 半合 <стихия>
🥉 Точечная возможность: <месяц/год Z> — 桃花 или одиночный 貴人
```

Для каждого пункта в **2-3 предложениях** объясни:
- какой паттерн (3-out-of-3 / 半合 / 桃花 / 貴人);
- какие конкретно ветви образуют слияние;
- **что это значит в жизни клиента** (карьера / отношения / творчество / финансы — выбирай по контексту вопроса).

### Шаг 5 — Признай ограничения

В финале добавь:

> «Это интерпретация по принципам школы Мастера ЭдоХа — через накопленные ветви и тройные слияния. По классической школе (Раймонд Ло / Yuan Hai Zi Ping) окнами считаются периоды Полезного Божества — это другой угол, и периоды могут не совпадать. Если хотите классическую версию — скажите.»

## Пример применения (карта Богдана)

**Карта:** 卯 (год Кролика), 酉 (месяц Петуха), 卯 (день Кролика), 亥 (час Свиньи). Дневной Мастер 丁.

**Шаг 1:** В тройке Дерева 亥卯未 присутствует 亥 (час) + 卯 (год+день, два). Это **3 ветви из 3** уже в карте — врождённый ресурс Дерева, но **未** нет приходящего.

**Шаг 2:** Сканируем приходящие 2026:
- Июль 2026 (乙未) — месяц Козы. ✓ Приходит 未 — четвёртая ветвь.
- Год Козы 2027 (丁未) — повторение на годовом уровне.

**Шаг 3 — паттерн:** В карте уже 3 ветви Дерева → приходит 未 → **3+1 → амплификация Дерева**. Это не классическое «открытие окна» (тройка и так замкнута), это **усиление врождённого ресурса** до пика.

**Шаг 4 — 桃花 проверка:** Богдан, Дневной Мастер 丁 (Огонь Инь). Карта дня = 卯 → 桃花 для 亥/卯/未 = **子** (Крыса). Приходящий год Крысы (бывает раз в 12 лет) был 2020. Следующий — 2032. На горизонте 2026-2027 точечного 桃花 нет.

**Шаг 5 — 天乙貴人:** Для 丁 (как и для 丙) благородные ветви — 亥 и 酉. У Богдана 亥 в часе, 酉 в месяце — **врождённый двойной 貴人**. Это значит **большинство окон в его карте идут под защитой**. Когда приходит 亥 или 酉 — активируется охранник.

**Ранжировка ответа Анастасии:**
```
🥇 Большое окно: июль 2026 (乙未) — приходит четвёртое Дерево. Усиление врождённого ресурса Дерева — рост, новые проекты, гибкие решения. Может быть пиковый месяц года для творческого выражения.
🥈 Среднее окно: ноябрь 2026 (己亥) — приходит вторая Свинья, активирует 亥 в часе + 天乙貴人 защиту. Хорошее время для долгосрочных планов и тайных проектов.
🥉 Точечная возможность: сентябрь 2026 (丁酉) — второй Петух усиливает 酉, врождённый 天乙貴人. Окно для статусных шагов, документов, формализации.
```

## Чего этот алгоритм НЕ покрывает

- **化氣格 трансформации** (когда стволы сливаются в новый элемент) — отдельный алгоритм
- **奇門遁甲** (Ци Мень Дунь Цзя) — другая система, не Ба Цзы
- **Конкретный день внутри окна** — окно даётся месяцем; день выбирается через 择日 (calendar selection)
- **Период такта удачи (大運)** — окно работает поверх такта, не вместо него; нужно проверить что текущий такт не Цзи Шэнь, иначе сила окна снижается

## Открытые вопросы для обсуждения с Богданом

1. Должен ли алгоритм отличать «открытие нового» (3-out-of-3) от «усиления существующего» (3+1 как в примере Богдана)?
2. 桃花 имеет смысл интерпретировать только для вопросов про отношения/творчество? Или универсально?
3. Когда давать топ-3 окон, а когда **только топ-1** (если юзер уточнил «самое благоприятное»)?
4. Если в горизонте ±2 года нет ни одного окна (карта изолирована от ближайших троек) — что отвечать? «Используйте микро-возможности» или «работайте на стабилизацию»?
