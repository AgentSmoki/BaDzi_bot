# БаЦзы-Бот — Мастер-документ проекта

> Telegram-бот для персональных AI-консультаций по системе Ба Цзы (四柱命理)
> с консультантом Анастасией

**Разработчик:** Богдан
**Дата старта:** март 2026
**Статус:** Этап документации

---

## Суть проекта

AI-бот в Telegram, который:
1. Рассчитывает карту Ба Цзы по данным рождения (Python, детерминированно)
2. Ведёт персональные консультации через AI-консультанта Анастасию (Claude Sonnet)
3. Монетизируется через подписку Pro (Free: 3 вопроса/день, Pro: безлимит)

## Ключевые решения

| Решение | Выбор | Почему |
|---------|-------|--------|
| AI-модель | Claude Sonnet | Баланс цены (~$0.01/вопрос) и качества |
| Расчёт карты | Python-код (не AI) | Детерминированный, мгновенный, без ошибок |
| Хранение данных | SQLite → PostgreSQL | Простота на старте, масштабирование потом |
| Хостинг | Railway | Простой деплой, как dzyn_bot |
| Монетизация | Free 3 вопроса/день + Pro подписка | Конверсия через демонстрацию ценности |
| Платежи | Telegram Payments / ЮKassa | Привычно для аудитории |

---

## Стек

- **Python 3.11+** — aiogram 3.x, SQLAlchemy 2.0 async, Anthropic SDK
- **Расчёт:** собственные таблицы + алгоритмы (солнечный календарь, 60-дневный цикл)
- **Геокодирование:** geopy + timezonefinder (город → таймзона)
- **Хостинг:** Railway (Procfile)
