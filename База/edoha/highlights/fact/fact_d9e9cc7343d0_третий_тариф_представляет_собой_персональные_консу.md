# Fact

третий тариф представляет собой персональные консультации для решения задач без теории по фиксированной оплате за час


## Доказательства

- 3 тариф и формат консультации. Первый тариф и формат консультации – это персональные консультации для решения задач и проблем клиента. Без объяснения, без обучения, без теории, только решение непосредственно задач за счет персональной консультации.
- В персональной консультации идет фиксированная оплата за час.
