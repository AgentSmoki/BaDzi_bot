## 5.2. Отношения и семья

### Что анализировать:
1. **Дворец Супруга (配偶宮)** — Дневная Земная Ветвь:
   - Какой элемент стоит? Какое Божество?
   - Есть ли столкновения с Дворцом Супруга?
   - Есть ли слияния (может указывать на близость или "поглощение" личности партнёром)

2. **Звезда Супруга:**
   - Для женщин: Прямой Чиновник (正官) = муж, 7 Убийца (七殺) = любовник/неформальный партнёр
   - Для мужчин: Прямое Богатство (正財) = жена, Склонённое Богатство (偏財) = любовница/неформальная партнёрша
   - Сколько звёзд супруга в карте? (Много = несколько отношений)
   - Где расположены? (Рядом с ДМ = рано, далеко = поздно)

3. **Совместимость с партнёром:**
   - Сравни карты обоих:
     - Дневные столпы — основная совместимость
     - Месячные столпы — совместимость в целях и амбициях
     - Слияния между картами — притяжение
     - Столкновения между картами — конфликт
   - Юн Шэнь одного = элементы в карте другого? (Если да — партнёр полезен)

4. **Благоприятные периоды для отношений:**
   - Такты и годы, когда приходит Звезда Супруга
   - Слияния с Дворцом Супруга
   - Приход Цветка Персика

### Важное ограничение:
```
"Я могу показать тенденции и потенциал совместимости, но отношения — это всегда двустороннее усилие. Карта показывает энергетическую предрасположенность, а не гарантию)"
```
